# 🌃 NEON SHADOW: Echo Runner

**NEON SHADOW: Echo Runner** is a high-octane, cyberpunk-themed endless runner built for Android. Navigate a futuristic cityscape as a neon-charged cyber ninja, vaulting over obstacles and running across floating platforms in a race for the highest score.

> [!IMPORTANT]
> **AI-Driven Development:** This entire project—including the core game engine, physics, collision systems, UI/UX design, and database architecture—was created **completely using Gemini (Google AI)**.

---

## 🚀 Features

- **Dynamic Stick-Man Animation:** Procedural, physics-based animations for running, jumping, and sliding with neon-pulse effects.
- **Cyberpunk Aesthetics:** High-contrast "True Black" AMOLED support with vibrant neon glow effects and glassmorphism UI.
- **Advanced Obstacle System:**
    - **Thorned Hazards:** Red neon spikes clearly indicate lethal collision zones.
    - **Multi-Level Running:** Large platforms allow for vertical gameplay and strategic positioning.
    - **Vaulting & Sliding:** Dynamic obstacles that require split-second timing.
- **RPG Progression:** Collectible power-ups (Shields, Speed Boosts) with visual state changes.
- **Offline High-Score System:** Local persistence using Room Database to track your top runs without an internet connection.
- **Performance Optimized:** 60 FPS guaranteed on mid-range devices using the Jetpack Compose Canvas API.

---

## 🛠 Tech Stack

- **Language:** Kotlin
- **UI Framework:** Jetpack Compose (Declarative UI)
- **Game Rendering:** Compose Canvas + `withFrameNanos` Game Loop
- **Architecture:** MVVM (Model-View-ViewModel)
- **Local Database:** Room Persistence Library
- **Dependency Management:** Version Catalogs (libs.versions.toml) + KSP
- **Navigation:** Jetpack Navigation Compose

## 🎮 How to Play

1. **Start:** Press "START RUN" from the Main Menu.
2. **Jump:** Tap the screen to jump. You can jump from the ground or while running on top of a platform.
3. **Slide:** Swipe down to slide under flying drones or low hazards.
4. **Collect:** Pick up glowing power-ups to gain temporary invulnerability or speed.
5. **Avoid:** Do **NOT** touch the red neon spikes (thorns).

---

## 🛠 Build Instructions

1. Clone this repository.
2. Open the project in **Android Studio (Ladybug or newer)**.
3. Ensure you have **JDK 17** or higher configured.
4. Build and run the `:app` module on an emulator or physical device (Android 7.0+ / API 24).
